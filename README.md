# T-SQL Practice Repository

A comprehensive collection of T-SQL learning materials, practice problems, and code examples from various sources.

## Directory Structure

### `/codes`
Contains hands-on SQL practice and problem-solving code organized by source:
- **book_practice_inside_tsql_querying/**: Practice exercises based on "Inside T-SQL Querying" book
- **hackerrank/**: HackerRank SQL challenges and solutions
- **leetcode/**: LeetCode SQL problems
- **StrataScratch/**: StrataScratch SQL interview problems
- **local_dbs/**: Scripts and configurations for local databases

### `/databases`
Database backup files:
- **Northwind.bak**: Northwind sample database backup
- **WideWorldImporters-Full.bak**: Wide World Importers sample database backup

### `/inside t-sql querying resources`
Complete resource files from "Inside Microsoft SQL Server 2005: T-SQL Querying" textbook by Itzik Ben-Gan:
- Chapter 01-09 files covering key T-SQL topics
- Appendix A with logic puzzles
- Database setup scripts (instnwnd.sql, instpubs.sql)

**Topics covered:**
- Logical and Physical Query Processing
- Query Tuning
- Subqueries, Table Expressions & Ranking Functions
- Joins and Set Operations
- Data Aggregation and Pivoting
- TOP and APPLY
- Data Modification
- Graphs, Trees, Hierarchies & Recursive Queries

### `/t-sql_code_books`
Additional SQL reference materials and code books:
- Joe Celko's SQL for Smarties
- SQL Practice: 57 Problems
- SQL Cookbook by Anthony Molinaro

## Purpose

This repository serves as a personal learning and reference collection for:
- Mastering T-SQL querying techniques
- Solving SQL interview and competitive programming problems
- Understanding advanced SQL concepts and optimization
- Building practical SQL skills through varied problem sets

## Getting Started

1. Restore sample databases from `/databases` using the .bak files
2. Run setup scripts (instnwnd.sql, instpubs.sql) to populate sample data
3. Explore practice problems organized by source
4. Reference textbook materials for deep dives into T-SQL concepts

- **Practice Platforms**: HackerRank, LeetCode, StrataScratch
